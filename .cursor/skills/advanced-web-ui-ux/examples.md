# Examples

## Hero Copy Upgrade

Before:
- Heading: `Digital solutions for your business`
- CTA: `Learn More`

After:
- Heading: `Turn website visitors into booked clients`
- Subheading: `Conversion-focused design and development for service businesses that need leads, not just pageviews.`
- CTA: `Book a free strategy call`

## CTA Microcopy

Use specific, low-friction CTA labels:

- `Start free trial`
- `Get instant quote`
- `See live demo`
- `Download pricing guide`

Avoid:

- `Submit`
- `Click here`
- `Learn more` (as primary CTA)

## Benefit Card Pattern

Use three concise benefit cards:

1. Outcome in 4-6 words
2. One sentence of supporting detail
3. Optional proof metric

Example:
- `Respond 2x faster to leads`
- `Automated follow-ups keep prospects engaged after first contact.`
- `Teams report 34% faster response times.`

## Form UX Pattern

- Keep fields minimal for first conversion step.
- Show helper text only where needed.
- Validate inline and explain errors in plain language.
- Confirm submission outcome with clear next step.

Example error:
- `Please enter a valid work email (name@company.com).`

## Section Flow Example

Recommended order:

1. Value-focused hero
2. Problem framing
3. Benefits
4. Social proof
5. FAQ
6. Final CTA
